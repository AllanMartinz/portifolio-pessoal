Projeto: site_acess_facul

O site_acess_facul é um projeto de desenvolvimento web criado com o objetivo de aplicar boas práticas de acessibilidade e responsividade. Ele foi desenvolvido para promover a inclusão digital, com foco em oferecer uma navegação acessível para pessoas com diferentes tipos de deficiência. Durante sua construção, foram adotadas diretrizes da WCAG (Web Content Accessibility Guidelines), utilizando HTML5, CSS3 e JavaScript para estruturar, estilizar e adicionar interatividade ao site. O layout também foi projetado para se adaptar a diferentes tamanhos de tela, garantindo uma boa experiência tanto em dispositivos móveis quanto em desktops.

Tecnologias utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • Visual Studio Code
  • Design Responsivo

Objetivo

O principal objetivo deste site é proporcionar uma experiência de navegação inclusiva e intuitiva. Para isso, foram implementados recursos como alto contraste, navegação por teclado, textos alternativos em imagens, estrutura semântica no HTML e design responsivo com media queries em CSS. Além disso, o projeto serviu como uma oportunidade prática de aprofundar meus conhecimentos nas tecnologias mencionadas, reforçando minha habilidade de construir interfaces acessíveis e centradas no usuário.
